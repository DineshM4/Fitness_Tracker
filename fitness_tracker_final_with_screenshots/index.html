Fitness Tracker

A full-stack workout logging app - FastAPI + Streamlit

What is it?

Fitness Tracker is a two-tier web application for logging workouts, tracking personal records, and visualizing your progress over time - all from the browser.

FastAPI - async-ready REST API with automatic OpenAPI docs at /docs. Validates every request with Pydantic models.
Streamlit - instant Python-powered UI. Custom CSS cards, tabs, sidebar form, and charts - no JavaScript needed.
SQLite - zero-config embedded database. Auto-created and seeded with demo data on first run.
Pandas - powers the dataframes and line charts for weight progression and workout history.

API Reference

Method Endpoint Description
POST /workouts Add a new workout
GET /workouts Retrieve all logs (optional ?exercise=)
GET /workouts/exercises List distinct exercise names
GET /workouts/stats Dashboard stats (streak, volume, heaviest)
GET /workouts/prs Personal records per exercise
PUT /workouts/{id} Update a workout by ID
DELETE /workouts/{id} Delete a workout by ID
DELETE /workouts Delete all workout logs

Interactive Swagger docs at http://127.0.0.1:8000/docs

Launch Instructions

1
Clone & enter the project
cd Fitness_Tracker
2
Create a virtual environment
python3 -m venv .venv
source .venv/bin/activate
3
Install dependencies
pip install -r requirements.txt
4
Start the backend
python app.py
→ http://127.0.0.1:8000
5
In a second terminal, start the frontend
streamlit run streamlit_app.py
→ http://localhost:8501
